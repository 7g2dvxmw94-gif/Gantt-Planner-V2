Questions fréquentes

Puis-je utiliser Gantly hors ligne? +
Gantly est une application web cloud. Une connexion Internet est nécessaire, mais vos données sont sauvegardées en temps réel et synchronisées partout.
Puis-je importer mes données existantes? +
Oui! Gantly importe les formats JSON, XML (Microsoft Project), Excel et CSV.
Quelle est la durée du trial Pro? +
Vous avez 14 jours gratuits du plan Pro. Aucune carte de crédit requise pour commencer.
Comment puis-je exporter mes projets? +
Gantly exporte en PDF et PowerPoint. Vous pouvez aussi télécharger vos données brutes au format JSON à tout moment.
