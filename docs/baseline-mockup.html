5 · Notes techniques d'implémentation

Store Structure de données

  • Chaque projet reçoit un tableau baselines[]
  • Chaque baseline : { id, name, createdAt, tasks: [{id, start, end, duration, progress}] }
  • activeBaselineId par projet
  • Sauvegarde dans localStorage (déjà existant)

Renderer Affichage

  • Barre baseline rendue sous la barre actuelle
  • Hauteur de ligne augmentée : 56px → 64px
  • Tag variance positionné en absolut
  • Toggle On/Off sans re-render complet

UX Interactions

  • Max 5 baselines par projet
  • Création instantanée (snapshot du store)
  • Activation d'une baseline = re-render Gantt
  • Colonnes BL dans Tableau : affichage conditionnel
